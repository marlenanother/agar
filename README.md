# Agar.io x Hunger Games (Browser)
- Клиент: Vite + TypeScript + Canvas, адаптивный UI.
- Сервер: Node.js + ws, авторитетная физика, лидерборд, экономика.

## Запуск
1) npm i
2) npm run dev (клиент)
3) cd server && npm i && npm run dev (сервер)
4) Открой http://localhost:5173

## Фичи
- Локально с ботами + онлайн через WebSocket
- Деление массы (G), передача массы союзнику (F)
- XP/уровни, валюта, магазин скинов, квесты, батлпасс
- Сессионный лидерборд (только реальные игроки), глобальный топ (без ботов)
- Адаптивный дизайн для ПК/мобилки

## Расширение
- Режимы: дуо/сквад, турниры, события, сужение зоны
- Бэкенд: подключить Redis/PostgreSQL, авторизацию, кланы
